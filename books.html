Books

I will not lie I am not a big fan of reading but I'll be sharing you amazing books for programming.

For different type of programming languages there are different types of programming books. Lets start for Python programming. I prefer Introduction to Programming using Python written by Y Daniel Liang for various of reasons. It has better questions that are really helpful for a first time programmer and also amazing content from scratch. Another book I'll suggest for Python is python Programming by Jhon M Zelle.

Java is an essential language for any programmer to reach a high level. The first book I'll recommend is Head First Java. This series of books have many versions for different programming languages. The books looks a bit silly but holds the core concepts that are fundamental for your advancement in that programming language. The second book I'll recommend is Java in Two Semesters. This book holds all the concepts and it's easier to follow.

The next language is going to be HTML, CSS, and Javascript. For This series I only appreciate the online resources like W3 school, Mozilla documentation. But one book I like is the Head first for Javascript and for the back end development like PHP.
